# rick_and_morty_api
Демонстрация Clean Architecture на примере мобильного приложения (Flutter + Dart).

## Learning Log
[14.11.2025, 16:56]:<br>*Чтобы улучшать свои навыки и создавать более надёжные, масштабируемые и поддерживаемые системы, мною было принято решение начать изучение Clean Architecture.*<br><br>
[14.11.2025, 17:11]:<br>*Понял, что суть Clean Architecture заключается в разделении приложения на несколько независимых слоев, каждый из которых должен выполнять свою, определенную задачу. Ключевая цель — это отделить пользовательский интерфейс и другие инфраструктурные компоненты от бизнес-логики. Таким образом, код становится более независимым и его легче тестировать в дальнейшем.*<br><br>
